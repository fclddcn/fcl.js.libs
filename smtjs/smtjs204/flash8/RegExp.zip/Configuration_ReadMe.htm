The Configuration Folders

The Configuration folders contain elements of the Flash MX authoring tool that can be customized. If you customize or extend the Flash MX product, you modify these files or add your source files to one or more of these folders. By familiarizing yourself with these folders, you can discover the interface for extensibility.

Warning: The files in the Configuration folders make up much of the Flash MX product. Do not modify, delete, or add to these files unless you know how to customize and extend Flash MX.

The Configuration folder exists in two separate locations depending on your Operating System and setup. The first location is relative to the Application. The second is stored in a location that is writable for the person who is currently the active user of your workstation. Depending on your operating system this will be in one of the following locations:

Windows:
98 and ME: C:\[Windows Directory]\Application Data\Macromedia\Flash MX

NT: C:\[Windows directory]\profiles\<user name>\Application Data\Macromedia\Flash MX

2000 and XP: C:\Documents and Settings\<user name>\Application Data\Macromedia\Flash MX

Mac OS:
Classic (Mac OS 8.x & above) Single-User: Hard Disk:System Folder:Application Support:Macromedia:Flash MX

Classic (Mac OS 8.x & above) Multi-User: Hard Disk:Users:<user name>:Documents:Macromedia:Flash MX

Mac OS X: Hard Disk/Users/<user name>/Library/Application Support/Macromedia/Flash MX

 

Application Level Configuration Directory
File/folder name Contents
HelpSWF Macromedia Flash content used within the Help menu.

 

User Level Configuration Directory
File/folder name Contents
authplay.dll External Macromedia Flash Player for use within the authoring tool (test movie, debug movie).
missfont.map Mapping file that contains fonts selected for substitution.
ActionsPanel

Files used to customize the Actions panel.

ActionsPanel.xml - configuration XML for contents of toolbox
AsCodeHints.xml - configuration XML for Code Hinting
AsColorSyntax.xml - configuration XML for Color Syntax highlighting
  CustomActions

Files used to add custom actions to the toolbox.
AnswersPanel

Content files used to drive the Answers panel and update functionality.
Color Sets

Color sets used by the color management system.
Components

Location of FLA source files that drive the Component panel.

HTML

HTML templates used for the publish command.

Importers

Files used to extend the importers available within Flash MX.

Keyboard Shortcuts

Specifies the custom keyboard shortcut and mnemonics sets available within Flash MX.

Libraries

Location of FLA source files that drive the Window > Common Libraries menu.

Panel Sets

Specifies the default, developer, designer and custom panel sets available within Flash MX.

Templates

Location of FLA source files that drive the File > New from Template... dialog.

WindowSWF

Macromedia Flash content used within the Window menu.
